<a href="https://declare-ai.org">declare-ai.org</a> - Declare How You Use Generative AI

This repo contains the source:

1. Install Docker
2. In a terminal:

   ```
   dx/start
   ```

3. When that completes, in another Terminal:

   ```
   dx/exec bin/setup
   dx/exec bin/build
   dx/exec bin/serve
   ```
4. View the site locally at `localhost:9998`
